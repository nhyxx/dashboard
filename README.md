## 대구-경북 진드기 매개 감염병 예측 대시보드

### 사용한 모델: XGBoost
    - 시군구별 농가 인구/고령층 인구 수, 감염병 발생자 수, 월 평균 강수량/기온 데이터 변수별 가중치 산출

### 데이터
    - KOSIS 국가 통계 포털: 시군구별 농가 인구/고령층 인구 수
    - 질병관리청 감염병 포털: 감염병 발생자 수
        - 렙토스피라증
        - 신증후군출혈열
        - 중증열성혈소판감증후군(SFTS)
        - 쯔쯔가무시증
    - 기상청: 월 평균 강수량/월 평균 기온
